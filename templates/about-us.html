ABOUT US

The issue of Education is paramount in the time of COVID-19. Therefore, we intend to create a Learning Management System(LMS) which aims to satisfy the void . Visualis Website version 1.0 is a web based LMS (Learning Management System) application which will allow users to pursue courses of their choice. Courses can be created and modified by Professors/ Mentors. Each course will have a week-wise video and a test to evaluate the student’s understanding. The Weekly Tests will help the Professors/Mentors to understand the student’s understanding and recommend him suitable paths to pursue. A course can continue across multiple weeks and have an optional test each week to assess students. Since, many colleges and universities find it difficult to conduct classes online, Visualis lets Professors create courses and upload the class content online and students can access the content. Blogs regarding the latest news in the education field or technology can be created and uploaded into the website.

OUR MISSION

Visualis uses Django framework as its backend. Django follows an MVT (Model-View-Template) Architecture. Loading the Template and its functionality from views can be taken care of using Django. The data will be stored in databases powered by PostgreSQL. The templates will be designed using HTML and styled using CSS. JS and jQuery will be used to provide basic animations to the HTML components. The basic functionality of the website includes: Student Login, Browsing Courses, Enrolling for a Course, Taking Tests and Completing the course. Admins can add, edit and remove courses, blogs and information about streams and branches.

OUR VISION

Our vision is to help discover the concealed potential of a student and guide him/her to build a successful career by making them star performers in their field of study.
